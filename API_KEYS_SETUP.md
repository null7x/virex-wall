# VIREX Wallpapers - API Keys Configuration

# ==========================================

#

# This file contains API keys for wallpaper sync from external sources.

# DO NOT commit this file to version control!

#

# To get your API keys:

#

# 1. UNSPLASH (https://unsplash.com/developers)

# - Create a free account at unsplash.com

# - Register a new application

# - Copy your "Access Key" (not Secret Key)

# - Free tier: 50 requests/hour (demo), apply for production for more

#

# 2. PEXELS (https://www.pexels.com/api/)

# - Create a free account at pexels.com

# - Request an API key

# - Free tier: 200 requests/hour, 20,000/month

#

# Add these keys to your local.properties file:

# UNSPLASH_ACCESS_KEY=your_unsplash_access_key_here

# PEXELS_API_KEY=your_pexels_api_key_here

#

# The app will work without these keys, but Auto Sync feature will be disabled.

# Example (replace with your actual keys):

# UNSPLASH_ACCESS_KEY=abc123def456ghi789jkl012mno345pqr678stu901vwx

# PEXELS_API_KEY=xyz987wvu654tsr321qpo098nml765kji432hgf109edc
